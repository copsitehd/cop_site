# COP — Centro Odontológico Palhoça

Landing page institucional do **Centro Odontológico Palhoça** (Palhoça/SC).
Especialistas em Reabilitação Oral, implantes, próteses, ortodontia e estética.

## Sobre o projeto

Site estático, leve e responsivo. Toda a página vive em um único arquivo `index.html`
(HTML + CSS + JS), e as imagens ficam em `assets/`.

## Estrutura

- `index.html` — a página completa
- `assets/` — logo, fotos da equipe, recepção, Espaço Kids e casos de antes & depois

## Deploy

Hospedado na **Vercel**. Cada commit na branch `main` publica o site automaticamente.
